## 使用方式

- GitHub Copilot Chat (VS Code)
   
将 `ezuikit-js-integration.md` 放在你当前开发项目的根目录（例如新建一个 docs/ 文件夹放进去）。
在 Copilot Chat 侧边栏对话框中，输入 #file 然后选择这个 .md 文件（或者直接拖拽文件到聊天框中）。
然后对 Copilot 说：
"根据这个 ezuikit-js 接入指南的文档，帮我在 React/Vue 项目中写一个带有全屏和截图功能的视频播放组件。" Copilot 就会完全基于这篇指南（特别是 8.2.0+ 的新 API）为你生成代码。

-  Cursor
  
把文件放在项目里。
在对话框中输入 @，然后输入 `ezuikit-js-integration.md`。
Cursor 会自动读取整个文件的内容作为“技能上下文”来回答你的编码问题。

- Kiro
   
在项目根目录下新建`.kiro/skills` 目录， 把 `ezuikit-js-integration.md` 放置在这个目录下
